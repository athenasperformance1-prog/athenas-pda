# ⚡ Athenas PDA — Plano de Ação Gamificado

App mobile-first com gamificação para acompanhar as metas da Athenas Marketing.

## 🚀 Como colocar no ar (5 minutos)

### Passo 1 — Subir o código pro GitHub
1. Crie uma conta no [GitHub](https://github.com) (se ainda não tem)
2. Clique em **"New repository"** (botão verde)
3. Nome: `athenas-pda` → Create repository
4. Faça upload de TODOS os arquivos desta pasta para o repositório

> **Dica rápida**: Na página do repositório vazio, clique em "uploading an existing file" e arraste toda a pasta

### Passo 2 — Deploy no Vercel (grátis)
1. Acesse [vercel.com](https://vercel.com) e faça login com sua conta GitHub
2. Clique em **"Add New Project"**
3. Selecione o repositório `athenas-pda`
4. Framework: **Vite** (deve detectar automaticamente)
5. Clique em **"Deploy"** — aguarde ~1 minuto
6. Pronto! Você receberá um link tipo: `athenas-pda.vercel.app`

### Passo 3 — Instalar no celular como app
**iPhone (Safari):**
1. Abra o link no Safari
2. Toque no ícone de compartilhar (quadrado com seta)
3. Role até **"Adicionar à Tela de Início"**
4. Toque em **"Adicionar"**

**Android (Chrome):**
1. Abra o link no Chrome
2. Toque nos 3 pontos (menu)
3. Toque em **"Adicionar à tela inicial"** ou **"Instalar app"**
4. Confirme

O app vai abrir em tela cheia, sem barra do navegador, igual um app nativo! 🎉

## 📱 Funcionalidades

- **Home**: Dashboard com faturamento, progresso e roadmap mensal
- **Semanal**: Checklist de 9 ações com animações de partículas e pontos
- **Plano**: 27 ações estratégicas em 3 fases (Fundação → Crescimento → Escala)
- **Funil**: Controle de leads com mudança de etapa por toque
- **Ranking**: Sistema de pontos com 5 níveis (Bronze → Lendário)

## 💾 Persistência

Todos os dados são salvos automaticamente no navegador (localStorage).
Os checks, dados mensais e leads ficam salvos mesmo fechando o app.
